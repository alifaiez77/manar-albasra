# منار البصرة — متجر الزيوت والفلاتر

موقع ومتجر إلكتروني لشركة **منار البصرة لتجارة الزيوت والفلاتر**، الوكيل المعتمد لعلامتي **Coral Oil** و **Shield** في جنوب العراق.

## المزايا
- تصميم سريالي بواجهات زجاجية (Glassmorphism) وثيم أبيض.
- خلفية متحركة مع التنقل: علبة زيت تنفتح وتسكب الزيت، وقطع محرك وعبوات تنساب وتتوزّع عبر الأقسام.
- متجر كامل: كتالوج 20 منتجاً، فلاتر حسب النوع والعلامة، وسلّة شراء تُحفظ محلياً مع إتمام الطلب عبر واتساب.
- متجاوب بالكامل مع الهاتف والأجهزة اللوحية وسطح المكتب.
- خط Al-Hurra العربي المضمّن.

## التشغيل
الموقع ثابت (HTML/CSS/JS) بلا أي اعتمادات. افتح `index.html` مباشرة، أو شغّل خادماً محلياً:

```bash
npx serve .
```

## البنية
```
index.html        الصفحة الرئيسية
assets/style.css  التصميم والحركة
assets/app.js     المتجر والسلة وحركة الخلفية
assets/fonts/     خطوط Al-Hurra
products/         صور المنتجات
images/ videos/   عناصر الخلفية والوسائط
```
